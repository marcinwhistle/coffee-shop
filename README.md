# Coffe Shop
Application for Coffe Shop written in Vanilla Javascript.
> Live demo [https://react-waiter-app.lemarczini21.repl.co/.](https://project-pizzeria.lemarczini21.repl.co)


## Technologies Used
- Vanilla JavaScript
- HTML
- CSS/SASS

## Features
- Render products using HandlebarsJS
- RWB
- WEB API
